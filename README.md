#### 本项目中利用到的第三方框架：</br>
- 项目富文本处理：YYKit</br>
- 布局：Masonry</br>  
- cell高度计算：UITableViewCell+HYBMasonryAutoCellHeight</br>
消息推送:</br>
- MQTT 和 WebSocket</br>
项目中可能会遇到的问题：滑动的流畅性，富文本显示问题，表情键盘封装以及弹出，项目的结构。</br>
